Reglas del juego

  • Comenzarás volteando una carta
  • Si la siguiente carta que volteas coincide, recibirás una alerta emergente y sumarás +1 a tu puntuación
  • Estas cartas luego desaparecerán
  • Si la siguiente carta que volteas no coincide, recibirás una alerta emergente indicándolo y las cartas se volverán a voltear
  • El juego continúa hasta que emparejes todas las cartas en el tablero
